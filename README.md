# Zepto Inventory SQL Analysis

## Project Overview
This project analyzes Zepto inventory data using MySQL to identify
stock availability, pricing issues, discounts, and category-level insights.

## Tools Used
- MySQL Workbench 8.0
- SQL
- CSV Dataset (Kaggle)

## Key Analysis
- Product count and data validation
- Out-of-stock product identification
- Category-wise pricing and discount trends
- High-discount and high-price product analysis

## Skills Demonstrated
- SQL querying
- Data validation
- Aggregations and filtering
- Business insight generation
